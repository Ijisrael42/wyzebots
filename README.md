# WyzeBots

Models
- WyzeBots
- Squads
- Tribes

## Features

- React v.17
- TypeScript v.4
- Material-UI v.5
- React Router v.6
- React Context
- React Hooks
- CSS-in-JS styles
- Responsive
- Create-react-app under the hood

## Quick Start

1. Get the latest version

```shell
git clone https://github.com/Ijisrael42/wyzebots.git WyzeBots
cd WyzeBots
```

2. Run

```shell
npm install
```

3. Run

```shell
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
